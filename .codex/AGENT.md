# 项目概览

这是一个前后端分离的金融资产问答系统，目标是把两类问题放进同一个聊天界面：

- 资产问答：查价格、看一段时间的涨跌趋势、分析价格异动可能对应的事件。
- 金融知识问答：解释金融术语、总结公司财报、基于本地知识库回答常见金融问题。

前端位于 `frontend/`，使用 React + TypeScript + Vite + Ant Design，核心是单页聊天界面，支持模型切换、流式输出和趋势图展示。后端位于 `backend/`，使用 FastAPI 提供聊天、资产行情、知识库重建、模型列表和 trace 查询接口。
